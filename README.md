# JavaBankAccount

This is a simple Java project that demonstrates how a basic bank account system works.  
It includes creating an account, depositing money, withdrawing money, and checking the balance.

## Features
- Create a bank account  
- Deposit and withdraw funds  
- Display current balance  
- Basic input/output handling  

## How to run
1. Open the project in IntelliJ IDEA (or any Java IDE)
2. Run `Main.java`
3. Follow the console instructions

## Technologies
- Java
- IntelliJ IDEA
